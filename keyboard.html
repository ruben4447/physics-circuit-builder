Keyboard / Mouse Control

General

Keyboard

  • a - Toggles the "Analyse" tab.
  • Delete - Prompt to delete the current file. If no file is open, prompts to close file. Behaviour varies when cursor is over a component / wire.
  • e - Toggles the "Extra Info" slider.
  • s - Prompt to save the file.
  • w - Toggles the "Wire Creation" slider.

Mouse

  • Left-Click - If creating a wire: Will cancel the wire insertion (the res wire will disappear).
    If inserting component: Attempt to place the component. Only successfull if cursor is over the circuit board.
  • Right-Click - If creating a wire: Create a "node" or "handle" on the red wire. Insertion will continue.

If over a component

Keyboard

  • Delete / Backspace - Prompt to delete the component.

Mouse

  • Left-Click - If "Create Wire" is enabled: Begin creating a wire (visually red). The red wire will follow your cursor. If left-click on another component, will attempt to connect the two.
  • Right-Click - Will trigger the "toggle" command. See About Components for more information on a specific component.
  • Scroll - Will trigger the "on scroll" command. See About Components for more information on a specific component.

If over a wire

Keyboard

  • Delete / Backspace - If over a node/handle: Prompt to delete the node/handle. Else, prompt to delete the wire.

Mouse

  • Left-Click - If over node (orange circle): Will drag the node while the left-mouse button is held. If released, the node will become static. This allows re-shaping of the wire.
  • Right-Click - If inserting a component: Split wire and place component in the middle.
    Else, Will create a node/handle at the given position.
